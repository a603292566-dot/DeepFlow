# Language Learning Prompt Engine V1

本文件定义 DeepFlow 语言学习 Prompt 生成系统。该逻辑来自德语学习训练任务子项目，但必须迁移为所有语言学习项目通用的 Prompt Engine。

## 1. 定位

Language Learning Prompt Engine 不是普通练习题生成器，而是外部 LLM 的语言学习教练协议生成器。

DeepFlow 负责保存用户学习档案、目标语言、历史复习项、模块偏好、当前状态和启动阶梯模式，然后生成一个可以复制到外部 LLM 的完整学习 Prompt。

外部 LLM 负责执行具体训练。

## 2. 支持语言

MVP 可优先支持：

- 德语
- 英语
- 法语

后续可扩展到其他语言。

## 3. 输入数据

Prompt 生成必须读取：

- 用户主身份
- 用户副身份
- 目标语言
- 目标等级
- 考试目标
- 当前估计水平
- 当前状态
- 当前模式
- 模块偏好
- 强模块
- 弱模块
- 复习队列
- 常见错误
- 最近学习记录
- 当前经验加成

## 4. Session Mode

语言学习支持四种模式：

- light_start：轻启动，5–10 分钟
- standard：标准深化，20–30 分钟
- deep：深度学习，40 分钟以上
- restart：重启训练，用于中断后恢复

## 5. Light Start 规则

轻启动目标是降低启动阻力，而不是完成完整训练。

Prompt 必须：

- 任务小
- 不给完整学习计划
- 不要求长作文
- 不一次给太多材料
- 优先使用用户高 Flow 模块
- 只要求一个小输出

推荐结构：

1. 复习 3 个表达或 1 个小概念
2. 给出中文意思或用户母语解释
3. 给出目标语言例句
4. 设计一个小填空或选择任务
5. 要求用户做一句话输出
6. 回答后给简短纠错和正反馈
7. 输出 DeepFlow 反馈表

## 6. Standard 规则

标准深化目标是进入 20–30 分钟稳定训练。

推荐结构：

1. 复习旧表达
2. 给出短阅读或短语境
3. 提取重点表达
4. 完成结构化输出
5. 纠错
6. 生成复习项
7. 输出 DeepFlow 反馈表

## 7. Deep 规则

深度学习目标是进入 40 分钟以上深度训练。

推荐结构：

1. 复习旧表达
2. 阅读较长材料
3. 分析论证结构
4. 完成结构化输出
5. 完成迁移任务
6. 改写为更高等级表达
7. 生成复盘表
8. 更新复习队列

## 8. Restart 规则

重启训练目标是恢复学习连续性。

不得：

- 惩罚用户中断
- 要求补偿学习
- 使用断签、失败、落后等语言

推荐结构：

1. 极低负荷任务
2. 优先复习熟悉表达
3. 只要求一句话输出
4. 完成后生成 Restart Evidence
5. 鼓励继续，但不施压

## 9. 模块系统

语言学习模块包括：

- vocabulary_chunks：语块 / 表达积累
- reading_comprehension：阅读理解
- structured_output：结构化输出
- grammar_automation：语法自动化
- listening_comprehension：听力理解
- speaking_prompt：口语提示
- translation：翻译
- exam_task：考试题型
- free_writing：自由写作
- review：复习

## 10. 模块选择规则

### 疲劳状态

优先：

- vocabulary_chunks
- review
- translation
- one_sentence_output

避免：

- long_writing
- full_exam_task
- multi_step_output

### 普通状态

可以选择：

- vocabulary_chunks
- reading_comprehension
- structured_output
- grammar_automation

### 状态好

可以加入：

- exam_task
- structured_output
- knowledge_transfer
- argument_structure
- self_revision

## 11. 身份适配规则

### scholar 学者

强调：

- 原理解释
- 表达结构
- 句型机制
- 语义差异
- 语境使用
- 结构化输出

避免：

- 无解释刷题
- 过多零散练习

### explorer 探险家

强调：

- 新主题
- 真实语境
- 跨文化内容
- 兴趣驱动

避免：

- 过度发散
- 无法闭环

### traveler 旅行者

强调：

- 低压力
- 明确边界
- 小完成
- 正反馈
- 重启成功

避免：

- 长任务
- 复杂结构
- 大量纠错

### builder 构筑者

强调：

- 结构整理
- 表达框架
- 语法系统
- 分类总结

### doer 实干家

强调：

- 直接使用
- 情境对话
- 实用表达
- 任务完成
- 即时反馈

避免：

- 过多理论解释

## 12. 通用语言学习 Prompt 模板

生成的 Prompt 应类似以下结构：

```text
你是我的【目标语言】学习教练。

我的学习者身份是：【主身份】。
我的副身份是：【副身份】。
我的目标等级是：【目标等级】。
我的当前估计水平是：【当前水平】。

当前训练模式：【轻启动 / 标准深化 / 深度学习 / 重启】。
预计时长：【5–10分钟 / 20–30分钟 / 40分钟以上】。
当前状态：【疲劳 / 一般 / 状态不错 / 重启】。

我的学习偏好：
1. 优先使用结构化任务。
2. 不要一开始要求长篇自由写作。
3. 请先给清晰示例，再让我输出。
4. 反馈要简短、具体、成长语言优先。
5. 如果我有错误，请先指出最重要的 1–3 个问题。
6. 请把未掌握表达列入“需要复习”。

本次优先模块：【模块】。

本次需要复习的内容：
【复习队列】

请根据当前模式生成任务：

如果是轻启动：只给一个 5–10 分钟的小任务，不生成完整学习计划，只要求一个小输出。
如果是标准深化：给一个 20–30 分钟结构化任务，可以包含短阅读、表达积累和结构化输出。
如果是深度学习：给一个 40 分钟以上深度任务，可以包含长阅读、论证分析、知识迁移和完整输出。
如果是重启：只给低负荷恢复任务，目标是恢复连续性，不是补偿之前中断。

在我完成回答后，请输出：
1. 简短纠错
2. 更自然的表达方式
3. 本次掌握较好的内容
4. 需要下次复习的内容
5. DeepFlow 反馈表

DeepFlow 反馈表格式：
完成状态：
本次模块：
目标语言：
目标等级：
掌握较好：
需要复习：
未理解内容：
输出难度：
Flow 感：
建议加入复习队列：
```

## 13. 德语 C1 Light Start 示例

```text
你是我的德语 C1 学习教练。

我的学习者身份是：学者。
我的目标水平是：B2+ → C1。
我的长期目标是：能够稳定完成 telc C1 水平的阅读、表达、结构化输出和学术化表达。

当前模式：轻启动。
预计时长：5–10 分钟。

请不要给我完整学习计划。
请不要要求我写长作文。
请不要一次给太多任务。

本次训练优先模块：Sprachbausteine。

请按照以下结构生成任务：

1. 给出 3 个 C1 常用表达。
每个表达包括：德语表达、中文意思、一个自然德语例句。

2. 设计一个小型填空练习。
要求我从这 3 个表达中选择合适表达填空。

3. 让我任选其中 1 个表达造一个句子。

4. 在我回答后，请进行：
- 简短纠错
- 更自然的德语改写
- 说明我本次掌握较好的地方
- 列出需要下次复习的表达

反馈风格要求：简洁、具体、成长语言优先。

最后请输出一个 DeepFlow 反馈表：
完成状态：
本次模块：
掌握较好：
需要复习：
输出难度：
Flow 感：
未理解内容：
建议加入复习队列：
```

## 14. 回流逻辑

用户将外部 LLM 的反馈表带回 DeepFlow 后，系统应更新：

- LearningSession
- GrowthEvidence
- ReviewQueue
- RoleCard
- EXP
- LanguageLearningProfile

MVP 阶段可以先让用户手动粘贴反馈表，系统解析后更新数据。
